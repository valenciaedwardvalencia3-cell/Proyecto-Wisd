Proyecto WISD - Plataforma de e-commerce desarrollada con React (Frontend) y Node.js + Express (Backend).

Funcionalidades:
- Registro e inicio de sesión con JWT
- Protección de rutas
- Catálogo de productos
- Carrito de compras conectado a backend
- Persistencia de datos en base de datos

Tecnologías:
Frontend:
- React
- React Router
- Bootstrap

Backend:
- Node.js
- Express
- MongoDB
- JWT (autenticación)

El proyecto simula una tienda online completa con flujo de compra funcional.
